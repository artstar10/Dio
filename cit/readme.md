# 🚀 Bootcamp CI&T - Do Prompt ao Agente

Este repositório documenta minha jornada no bootcamp da CI&T em parceria com a DIO, com foco na aplicação prática de Inteligência Artificial no desenvolvimento de software.

---

## 🎯 Sobre o Bootcamp

- ⏱️ Carga horária: 35 horas
- 🤖 Foco: IA aplicada ao desenvolvimento
- 🧠 Do básico até agentes autônomos
- 💻 Projetos práticos e desafios reais
